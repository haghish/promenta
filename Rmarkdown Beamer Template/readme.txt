This is a beamer template developed for PROMENTA research center at University of Oslo. 
Please note that this template is NOT associate with the department and by no means is 
official. However, colleagues are welcome to use the template within R or LaTeX programs 
to generate their own slides. 

Use the MAIN.Rmd if you wish to write with Markdown within R. Use MAIN.Rnw if you wish to write with LaTeX within R. If you like to write comepletely outside R, use the example.tex
to create your own document.
